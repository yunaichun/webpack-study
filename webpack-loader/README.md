## 简介

webpack 自定义 loader.

## 项目启动

```bash
# 安装依赖
npm install

# loader 加载顺序查看
npm run test:loader-sequence

# loader-runner 工具测试自定义 raw-loader 是否成功运行
npm run test:raw-loader

# loader-runner 工具测试自定义 sprite-loader 是否成功运行
npm run test:sprite-loader
```
